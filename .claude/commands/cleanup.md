# 代码清理

扫描项目中的代码问题并自动修复：

## 检查项

1. **未使用的 using 声明** - 删除未引用的命名空间导入
2. **重复的 using 声明** - 删除重复导入
3. **using 排序** - 按字母顺序排列
4. **空白行规范** - 文件末尾单个空行
5. **尾随空格** - 删除行尾空格

## 执行方式

1. 使用 Grep 扫描 `src/**/*.cs` 文件
2. 识别问题文件
3. 使用 Edit 工具自动修复
4. 输出修改摘要

## 排除目录

- `**/obj/**`
- `**/bin/**`
- `**/*.g.cs`（生成的文件）
